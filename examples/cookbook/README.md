# Cookbook Examples

SDK versions of [Your First Memory in 5 Minutes](https://docs.evermind.ai/cookbook/quickstart).

## Prerequisites

```bash
pip install evermemos -U
export EVERMEMOS_API_KEY="your_api_key"
```

## Examples

| File | Corresponds to |
|------|---------------|
| `quickstart_sync.py` | Steps 1-3 (Store → Wait → Search), sync SDK |
| `quickstart_async.py` | Steps 1-3, async SDK |
| `quickstart_complete.py` | "Complete Working Script" section (user_bob) |

## Run

```bash
python quickstart_sync.py
python quickstart_async.py
python quickstart_complete.py
```
